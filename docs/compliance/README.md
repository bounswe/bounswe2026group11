# Compliance Documentation

This folder documents Social Event Mapper's accessibility and application security posture. It records the standards used by the project, the implementation evidence in each surface, the configuration that supports those controls, the verification commands used for regression coverage, and known limitations.

This documentation is project compliance evidence. It is not a formal third-party certification.

## Documents

- [Standards Scope](./standards-scope.md): targeted W3C/WCAG and OWASP references.
- [Accessibility](./accessibility.md): WCAG 2.1 AA implementation evidence for the web and mobile surfaces.
- [Security](./security.md): OWASP Top 10 and ASVS-aligned backend, frontend, and mobile security controls.
- [Configuration](./configuration.md): hardening-related configuration and defaults.
- [Verification](./verification.md): automated checks and the commands used to verify the implementation.
- [Limitations](./limitations.md): known gaps and conditions required before broader conformance claims.
