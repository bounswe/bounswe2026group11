# Standards Scope

Social Event Mapper targets the following practical compliance baselines.

## Accessibility

- W3C Web Content Accessibility Guidelines 2.1, Level AA: web accessibility baseline for perceivable, operable, understandable, and robust interfaces.

## Application Security

- OWASP Top 10:2025: application risk categories used as the primary security checklist.
- OWASP ASVS 5.0.0: verification vocabulary for authentication, access control, input validation, error handling, security headers, and operational controls.

## Certification Status

The documents in this folder provide implementation evidence for the project. They do not represent a formal third-party certification or a complete legal compliance claim.
